# 🔴🟡 四子棋 (Connect Four) - Windows Forms App

這是一個使用 C# 與 Windows Forms (.NET) 開發的經典四子棋桌面遊戲。
透過 GDI+ 繪圖技術，打造了平滑且無閃爍的視覺體驗，並支援「本地雙人對戰」以及「單人對戰 AI」兩種遊玩模式。

## ✨ 遊戲特色 (Features)

* 🎮 **雙重遊戲模式**：
  * **本地雙人 (PvP)**：與朋友在同一台電腦上輪流操作，支援自訂玩家名稱。
  * **單人對戰 (PvE)**：挑戰內建的 AI 系統（具備基礎防守與隨機落子邏輯）。
* 🎨 **客製化平滑 UI**：
  * 放棄傳統的 Button 拼湊，全面採用 **GDI+** 繪製棋盤與棋子。
  * 支援 **反鋸齒 (Anti-Alias)** 與 **雙緩衝 (Double Buffering)**，畫面順暢不閃爍。
  * 直覺的**滑鼠懸停 (Hover) 高光提示**，清楚顯示即將落子的直行。
* 🧠 **即時勝負判定**：高效的演算法，每次落子後即時檢查水平、垂直、正反對角線的 4 連線狀態。
* 🎵 **音效回饋**：每次落子皆有音效回饋，提升遊戲打擊感。
* 🗂️ **多表單架構**：具備獨立的「主選單介面」與「遊戲畫面」，提供完整的軟體體驗。

## 🚀 如何執行 (How to Run)

### 系統需求
* Windows 作業系統
* [Visual Studio](https://visualstudio.microsoft.com/) (包含 ".NET 桌面開發" 工作負載)
* .NET Framework 或 .NET Core / .NET 5+ (依專案設定而定)

### 安裝與啟動步驟
1. 將此專案 Clone 到本地端：
   ```bash
   git clone [https://github.com/您的帳號/您的專案名稱.git](https://github.com/您的帳號/您的專案名稱.git)使用 Visual Studio 打開專案中的 .sln (方案檔)。

2.按下 F5 或點擊 「開始」 編譯並執行遊戲。

## 🕹️ 遊玩說明 (How to Play)
在主選單選擇「本地雙人對戰」或「單人模式」。

雙人模式下，可於左上方文字框輸入自訂的玩家名稱。

移動滑鼠，棋盤會以深灰色底色提示目前選擇的直行 (Column)。

點擊滑鼠左鍵使棋子落下。

最先將 4 顆同色棋子 連成一線（水平、垂直、或斜線）的玩家即可獲勝！

若棋盤填滿且無人連線，則判定為平手。

遊戲結束後，點擊畫面任意處或「重新開始」按鈕即可開啟新局。
